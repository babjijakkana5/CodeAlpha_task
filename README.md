# Basic Calculator in C Using Switch Case

## 📌 Project Overview

This project is a simple **Basic Calculator** developed in the **C programming language** using the **switch-case** statement. It performs basic arithmetic operations based on the user's choice.

## ✨ Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* User-friendly menu
* Input validation for division by zero

## 🛠️ Technologies Used

* C Programming Language
* GCC Compiler (or any C compiler)

## 📂 Project Structure

```
Calculator/
│── calculator.c
└── README.md
```

## 🚀 How to Compile and Run

### Compile

```bash
gcc calculator.c -o calculator
```

### Run

```bash
./calculator
```

**For Windows (MinGW):**

```bash
calculator.exe
```
